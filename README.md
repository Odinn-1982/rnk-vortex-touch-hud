## Installation

1. In Foundry VTT, go to **Add-on Modules**
2. Click **Install Module**
3. Paste this URL: (your-download-url)
4. Click **Install**
5. Enable the module in **Manage Modules**
6. Refresh your browser

## Usage

The Touch HUD automatically appears on touch-enabled devices. Customize behavior through module settings.

### Touch Controls

- **Center Circle** - Primary action (attack, interact, etc.)
- **D-Pad** - Movement and camera control
- **Right Buttons** - Quick actions and spell casting
- **Swipe Gestures** - Switch between panels

## Configuration

Access settings through Foundry's module settings:

- Enable/disable HUD on startup
- Adjust button size for screen
- Customize layout (compact, standard, expanded)
- Audio feedback toggle
